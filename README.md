## üöÄ Installation

### From the Chrome Web Store
https://chromewebstore.google.com/detail/flow-timer-pomodoro/lhjchnebampfjcbamemnhcbbieldilfj?authuser=0&hl=en

### From Source (for Developers)

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/flow-timer-pomodoro.git
    ```
2.  **Open Chrome and navigate to the Extensions page:**
    Go to `chrome://extensions` or select `Menu > Extensions > Manage Extensions`.

3.  **Enable Developer Mode:**
    Click the toggle switch at the top right corner to enable "Developer mode".

4.  **Load the extension:**
    Click the "Load unpacked" button and select the directory where you cloned the repository.

5.  The Flow Timer icon should now appear in your browser's toolbar!

## üí° How to Use

1.  **Start a Session**: Click the extension icon in your toolbar and press the "Start Pomodoro" button. The timer will begin, and your blocked sites will become inaccessible.
2.  **Add Blocked Sites**: You can quickly add a site to your blocklist directly from the popup or manage the full list in the settings.
3.  **Customize Settings**:
    *   Click the settings gear icon (‚öôÔ∏è) in the popup or right-click the extension icon and select "Options".
    *   In the options page, you can:
        *   Adjust the duration of work, short break, and long break periods.
        *   Manage your blocked sites and whitelist.
        *   Change the notification sound or upload your own.
